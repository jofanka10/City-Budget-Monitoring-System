# Background Image Setup

## Instructions

1. Place your `Bg_pweb.jpg` file in the `frontend/public/` directory

2. The file path should be: `frontend/public/Bg_pweb.jpg`

3. The background image will automatically be applied to the application

## If you don't have the image yet:

The application will still work, but you'll see a default background. Once you add `Bg_pweb.jpg` to the `public` folder, refresh your browser to see the new background.
